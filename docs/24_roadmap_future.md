# Roadmap

## Estado final atual

A versão 1.0.0 está fechada como app Android local com APK/AAB release, orientação em retrato, análise de links, histórico, terminal, navegador seguro, acessibilidade opcional e VPN local.

## Possíveis próximos passos

Estes itens não estão implementados e só devem entrar no escopo se forem priorizados:

- Publicação na Play Store.
- Política de privacidade formal para distribuição pública.
- Migração opcional para banco local mais robusto.
- Backup/exportação mais completo para usuário final.
- Melhor cobertura de QR real com leitura de imagem/código.
- Reputação online com backend próprio, se necessário.
- Pipeline CI/CD.
- Crash reporting local/remoto com consentimento.

## Itens removidos do escopo atual

- Dashboard web.
- Conta de usuário.
- Sincronização em nuvem.
- API remota.
