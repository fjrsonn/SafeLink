# Release e Distribuição

## Versão final

- Version name: `1.0.0`.
- Version code: `100`.
- Flavor final: `full`.

## Artefatos

- `dist/SafeLink-1.0.0-full-release.apk`.
- `dist/SafeLink-1.0.0-full-release.aab`.

## Assinatura

- Keystore: `dist/safelink-release.keystore`.
- APK verificado com `apksigner`.
- Esquema v2 válido.

## Instalação por APK

1. Transferir o APK para o dispositivo.
2. Permitir instalação de fontes desconhecidas, se necessário.
3. Instalar o APK.
4. Abrir SafeLink.
5. Configurar as camadas desejadas.

## Observação sobre Play Protect

Apps instalados fora da Play Store podem exibir alerta do Play Protect. Para distribuição pública, o caminho recomendado é publicar pela Play Store com revisão formal.
